## 第一步：提取昨日对话

连接数据库，查询昨天的所有session和messages：

```sql
.headers off
.mode column
.width 50 20 20 200

-- 找出昨天的sessions
WITH yesterday_sessions AS (
  SELECT id, source, started_at, ended_at, message_count, title
  FROM sessions
  WHERE started_at >= date('now', '-1 day') AND started_at < date('now')
  ORDER BY started_at DESC
)
SELECT id, source, datetime(started_at, 'unixepoch'), message_count, title
FROM yesterday_sessions;
```

对每个session，查询其messages：
```sql
SELECT 
  datetime(timestamp, 'unixepoch') as time,
  role,
  substr(content, 1, 500) as content_preview
FROM messages
WHERE session_id = '<SESSION_ID>'
ORDER BY timestamp ASC;
```
